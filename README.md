dotfiles
========

## Prerequisites
Install all of these using your OS's package manager

* git
* vim
* mutt
* lynx
* terminator
* wmctrl
* tree
* grc
* universal-ctags
* Yandex Disk

## Setting up the environment
mkdir -p ~/bin && cd ~/bin
git clone git@github.com:EugeneSqr/dotfiles.git
./setup.sh
