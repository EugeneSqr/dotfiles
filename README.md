dotfiles
========

## Prerequisites
Install all of these using your OS's package manager

* bash-completion
* git
* vim
* mutt
* lynx
* mupdf
* terminator
* wmctrl
* tree
* grc
* fzf
* fd
* bat
* Yandex Disk

## Setting up the environment
```
mkdir -p ~/Documents && cd !$
git clone git@github.com:EugeneSqr/dotfiles.git && cd dotfiles
./setup.sh
```
